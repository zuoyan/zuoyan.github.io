1 Why using password manager

Not everyone agrees that a password manager is required, and only very little people using a password manager, mostly, in a wrong manner.

With password manager, you can use different passwords for different sites, different banks. If one site explosure you password, or, a hacker sniffed your password, the other passwords are still safe.

In the early ages, most site transfer passwords in plain text format, a simple sniffer program can be written to capture and analysis the datagrams containg the password. To be honest, I wrote one, and successfully signed in to others' BBS.

With password manager, you can use longer passwords. longer passwords is safer than short one, especially again a brute attack.

With password manager, you can forget your passwords. Only one local password is required. But this also introduce some inconvenience. Inconvenience is the reason of most people don't using a password manager. I also suffer from it. I can not sign in of my all online account on other's computers. I can not pay online, because I forget all cards' passwords.

So using password manager for safe, don't using password manager for conveniences.
